# 🧭 浙江高中物理主要学习内容（按必修 + 选修分类）

## ✅ **必修内容（全国统一）**

所有学生都要学：

### **1. 运动学**

* 质点模型
* 速度、加速度
* 匀变速直线运动
* 探究速度与时间、位移与时间、加速度与时间的关系
* 自由落体
* 曲线运动 & 合成分解（重要基础）

### **2. 力与机械**

* 牛顿三大定律
* 共点力平衡
* 滑动摩擦力
* 运动的合成与分解
* 圆周运动基本模型
* 万有引力与航天（必修中为基础引入）

### **3. 能量与动量**

* 功与功率
* 动能定理
* 机械能守恒
* 冲量与动量
* 动量守恒定律
* 碰撞问题（浙江高考必考）

### **4. 电学基础**

* 电流、电压、电阻
* 欧姆定律
* 并联串联电路
* 电功、电功率
  （这一部分较简单）

# 🔍 **选考物理（浙江选考科目重点）**

浙江物理的核心难点全部在选考内容，包括大量计算与复杂情境题。

## 🌟 **1. 力学进阶（侧重）**

* **圆周运动深化**
  向心力来源分析、临界速度问题、圆环、过山车、锥摆等
* **万有引力与天体运动（重点）**
  人造卫星、行星运动、逃逸速度
* **振动与波（大题常考）**
  简谐振动方程、能量、相位
  波的图像、叠加、干涉、衍射
* **机械波实验与多步推导题**

## ⚡ **2. 电磁学（浙江最重头内容）**

浙江特别喜欢考电磁学，且难度高。

* **电场**：电场强度、电势、电势能、带电粒子运动（直线/圆周/组合）
* **磁场**：洛伦兹力、带电粒子在磁场中的圆周运动、螺旋运动
* **电磁感应**：楞次定律、法拉第电磁感应定律、自感、互感
* **交变电流**：正弦交流、有效值、变压器
* **复杂电路分析**：
  多个回路、电表内阻、电路动态问题（浙江特色）

## 🔭 **3. 光学（中等难度）**

* 反射与折射规律
* 几何光学（成像）
* 光的波动（干涉、衍射）
* 光电效应（简单介绍）

## 💻 **4. 近代物理（简单）**

* 原子结构
* 能级跃迁
* 核反应基本公式
